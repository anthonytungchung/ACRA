# ACRA Weekly Report Standard v1.0

Every weekly report must contain:

1. Executive Summary
   - Total Stake
   - Total P/L
   - ROI
   - Strike Rate
   - Points P/L

2. Performance Charts
   ✓ Daily P/L
   ✓ Cumulative Equity Curve
   ✓ Rating Performance
   ✓ Win/Loss Breakdown

3. Biggest Winners

4. Biggest Losers

5. Framework Compliance

6. Sportily Review
   - What Sportily got right
   - What Sportily got wrong

7. Independent Analysis Review
   - What ACRA got right
   - What ACRA got wrong

8. Lessons Learned

9. Rulebook Changes

10. Watchlists
    - Horses
    - Trainers
    - Jockeys

11. Concrete Adjustments for Next Week

12. Overall Grade
